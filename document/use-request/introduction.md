# useRequest

基于 `Vue Composition API` 封装的请求 Hook，支持自动状态管理、防抖/节流、轮询、缓存、数据格式化等特性，适用于返回类型是 [ResponseContent](/api-reference/common-type/response-content.md) 的异步数据请求。

[具体 Api](/api-reference/hooks/use-request/home.md)
## 特性

* 响应式数据
* 自动请求/手动请求
* 节流、防抖请求
* 请求错误自动重试
* 轮询请求
* 缓存 & SWR
* 延迟/持续 loading
* 聚焦页面时自动重新请求
* 自定义插件
